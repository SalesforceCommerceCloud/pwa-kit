# PWA Kit Developer Tools

A command-line tool to develop, build, and deploy PWA Kit projects.

## Requirements

-   Node 16.11 or later
-   npm 8 or later

To see all the available commands, run:

```bash
npx @salesforce/pwa-kit-dev --help
```
