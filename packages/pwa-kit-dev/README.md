# PWA Kit Developer Tools

A command line interface tool to develop, build, and deploy PWA Kit projects.

## Requirements

```
  Node ^14.x
  npm ^6.14.4
```

To see all the available commands, run:

```bash
npx pwa-kit-dev --help
```
