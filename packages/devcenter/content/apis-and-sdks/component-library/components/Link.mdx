whether the component renders a react-route link or an anchor tag.

### Basic example

```jsx react-live=true
<Link href="http://mobify.com/">Mobify</Link>
```

### With child elements

```jsx react-live=true
<Link href="http://mobify.com/">
    <i>MOBIFY!</i>
</Link>
```

### With prop text

```jsx react-live=true
<Link href="http://google.com/" text="Go to Google" />
```

### With no href

```jsx react-live=true
<Link text="No URL provided" />
```

### With an onClick handler

```jsx react-live=true
<Link
    text="Click me!"
    onClick={(e) => {
        alert('Clicked'), e.preventDefault()
    }}
/>
```

### With hover (onEnter and onLeave) handlers

```jsx react-live=true
class StatefulExample extends React.Component {
    constructor() {
        this.state = {
            isHovering: false
        }
    }

    hoverOn() {
        this.setState({isHovering: true})
    }

    hoverOff() {
        this.setState({isHovering: false})
    }

    render() {
        return (
            <Link
                text={this.state.isHovering ? 'Hovering! ðŸ­' : 'Hover Me!'}
                onMouseEnter={this.hoverOn.bind(this)}
                onMouseLeave={this.hoverOff.bind(this)}
            />
        )
    }
}
```
