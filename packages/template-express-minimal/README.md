# Express Minimal

A bare-bones project template for apps running on
Managed Runtime.

You can use the template as a starting point to:

-   Build simple Express apps.
-   Develop APIs.
-   Run different frameworks (other than React) on Managed Runtime.

## Setup

```bash
npm ci
npm start
```

## Deploy

```bash
npm run push
```
